1. Introdução

O projeto "Fast Food Delivery App" é um aplicativo móvel desenvolvido para facilitar a experiência de alimentação dos usuários, permitindo que eles façam pedidos de fast food de maneira rápida e eficiente. Com a crescente demanda por soluções de entrega, este aplicativo visa proporcionar uma interface amigável e intuitiva, onde os usuários podem explorar uma variedade de opções, visualizar menus, personalizar pedidos e realizar pagamentos de forma segura.
O objetivo principal do aplicativo é otimizar o processo de pedido e entrega, reduzindo o tempo que os usuários gastam na escolha e compra de suas refeições. Além disso, o aplicativo oferece recursos como rastreamento de pedidos em tempo real, promoções exclusivas, garantindo que os usuários tenham uma experiência satisfatória e conveniente.
Com este projeto, buscamos não apenas atender às necessidades alimentares dos usuários, mas também transformar a forma como eles interagem com o serviço de delivery, tornando-o mais acessível e eficiente.

5. Instruções de Uso
5.1. Instalação
1.	Clone o repositório:
git clone https://github.com/andrefranziner/mcking
2.	Navegue até o diretório do projeto:
cd  
3.	Instale as dependências:
npm install @react-navigation/native
npm install @react-navigation/Stack
npm install react-native-screens react-native-safe-area-context
npm install react-native-vector-icons
npm install styled-components
npm install react-native-paper
    
5.2. Execução
Para iniciar o servidor, execute:
npm run web
